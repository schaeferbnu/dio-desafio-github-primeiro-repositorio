Principais comandos Git

- git add: adiciona arquivo ao index

- git commit: grava as mudanças ao repositório local

- git init: cria um repositório vazio e inicia nele

- git push: envia um repositório ao GitHub

- git pull: incorpora o repositório remoto ao local
 
- git status: informa o status da tree em uso
